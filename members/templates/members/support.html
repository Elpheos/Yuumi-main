Questions fréquentes

Appuyez sur "Mon compte" puis "S'inscrire". Renseignez votre nom d'utilisateur, votre adresse e-mail et un mot de passe. La création de compte est gratuite.
Sur la page d'un commerce, appuyez sur le bouton ❤️ "Ajouter aux favoris". Vous devez être connecté pour utiliser cette fonctionnalité. Retrouvez ensuite vos favoris dans "Mon compte → Mes favoris".
Lors de votre première connexion avec vos identifiants, l'application vous propose d'activer la reconnexion biométrique. Acceptez la proposition pour vous connecter rapidement les prochaines fois avec votre empreinte ou Face ID.
Sur la page du commerce concerné, appuyez sur "Suggérer une modification" et décrivez l'erreur. Notre équipe vérifiera et mettra à jour les informations dans les meilleurs délais.
Sur la page de votre commerce, appuyez sur "Prenez le contrôle de votre page". Vous devez être connecté. Notre équipe vous contactera après vérification pour valider la revendication.
Rendez-vous dans les Paramètres de votre téléphone → Applications → Yuumi → Notifications, et désactivez-les. Vous pouvez également les désactiver depuis les réglages de notifications de votre appareil.
Pour supprimer votre compte, contactez-nous par e-mail à contact@yuumi-shop.com en indiquant votre nom d'utilisateur. Nous traiterons votre demande sous 48h conformément au RGPD.
